<h2>Sobre mim</h2>

<img align="right" width="280" src="https://raw.githubusercontent.com/abhisheknaiidu/abhisheknaiidu/master/code.gif" alt="coding gif" />

<p>Desenvolvedor Web com experiencia pratica em projetos completos, do front-end ao back-end, incluindo infraestrutura, containers e automacoes.</p>

- Aplicacoes web modernas com **React**, **TypeScript** e **Next.js**
- APIs REST com **Node.js** e integracao com **Oracle Database**
- Ambientes com **Docker**, **Linux** e deploy em producao
- Automacoes com **n8n** integrando WhatsApp, APIs e bancos de dados
- Foco em **performance**, **escalabilidade** e **experiencia do usuario**
